
for auto run python file follow the step on terminal

1> sudo nano /home/pi/.bashrc
2> put your all source file in /home/pi/ dir
3> Go to the last line of the script and add:
	echo Running at boot
	sudo python /home/pi/sample.py
4> save and exit
5> sudo reboot
